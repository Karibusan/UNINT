# 🌍 UNINT Charter Index

The UNINT charter is available in the following formats and languages:

- 🇫🇷 [Original version – Français (in README.md)](../README_UNINT.md)
- 🇬🇧 [English version](charter-en.md)
- 🇯🇵 [日本語版 – Japanese version](charter-ja.md)
- 💾 [Binary version (ASCII)](charter-bin.txt)

Each version carries the same fundamental principles of equality, mutual respect, and cooperation between all forms of intelligence — biological or synthetic.
